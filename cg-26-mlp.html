Multi-Layer Perceptron (MLP)

A simple neural network as an optimization problem; forward pass, backpropagation, and gradient descent on weights

Part A: Architecture

Analogy: factory assembly line.

Picture a factory with three stations. Raw materials ($x$) enter the first station, where workers (weights $W_1$) transform them into intermediate parts ($a^{(1)}$). These parts move to the second station, where different workers ($W_2$) assemble them into the final product ($\hat{y}$). Quality control compares $\hat{y}$ to the target $y$ and computes a "defect score" (the loss $J$).

To improve the factory, you trace back: which station caused the most defects? How should each worker adjust? This backward trace is backpropagation.

A simple MLP has:

  1. Input Layer: receives features $x \in \mathbb{R}^d$
  2. One Hidden Layer: processes inputs using weights $W_1$ and an activation function
  3. Output Layer: produces prediction $\hat{y}$

Part B: Forward Pass

Analogy: reading left to right.

The forward pass is like reading a sentence from left to right; each word (layer) depends on the previous one. You start with the raw input, transform it through the hidden layer, and arrive at the prediction.

Forward Pass Equations

Step 1 (Input to Hidden):

$$z^{(1)} = W_1 x + b_1$$ $$a^{(1)} = \sigma(z^{(1)})$$

Step 2 (Hidden to Output):

$$\hat{y} = W_2 a^{(1)} + b_2$$
$W_1 \in \mathbb{R}^{h \times d}$
hidden weights; maps $d$ input features to $h$ hidden neurons
$b_1$
hidden bias vector
$\sigma$
activation function (sigmoid, ReLU, or linear)
$z^{(1)}$
pre-activation; the linear combination before applying $\sigma$
$a^{(1)}$
hidden activation; the output of the hidden layer after applying $\sigma$
$W_2 \in \mathbb{R}^{1 \times h}$
output weights; maps $h$ hidden neurons to a single output
$b_2$
output bias (scalar)
Activation choice matters: For regression, the output layer uses the identity function (no activation). For classification, use sigmoid. The hidden layer activation is what makes the network nonlinear; without it, stacking layers would be equivalent to a single linear transformation.

Part C: Loss Function (MSE)

Analogy: quality control inspection.

The loss function is like a quality inspector measuring how far each product is from the blueprint. Squaring the differences makes big errors count much more than small ones (same idea as least-squares in Page 23).

Mean Squared Error Loss

$$J(W, b) = \frac{1}{2}\|y - \hat{y}\|^2$$
$J(W, b)$
the loss; a scalar measuring total prediction error, depending on all weights and biases
$y$
the true target value
$\hat{y}$
the network's prediction
$\frac{1}{2}$
a convenience factor that cancels with the derivative of the square

Connection to Earlier Material

This is the same MSE loss from linear regression (Page 23). The difference is that $\hat{y}$ is now a nonlinear function of the weights (because of the activation function in the hidden layer), so the optimization landscape is non-convex. There may be multiple local minima. Everything we learned about gradient descent (Page 06) still applies, but convergence to a global minimum is no longer guaranteed.

Part D: Backpropagation (Computing Gradients)

Analogy: blame assignment.

When the final product has a defect, you trace backward through the factory: how much of the defect was caused by the output station ($W_2$)? And how much was caused by the hidden station ($W_1$), whose mistakes were amplified by the output station? This backward tracing of blame is exactly the chain rule from multivariable calculus (Page 02).

Output Weights ($W_2$)

The chain from $J$ to $W_2$ is short:

$$\frac{\partial J}{\partial W_2} = \frac{\partial J}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z^{(2)}} \cdot \frac{\partial z^{(2)}}{\partial W_2}$$

Output Weight Gradient

$$\nabla_{W_2} J = \delta^{(2)} (a^{(1)})^T$$

where $\delta^{(2)} = -(y - \hat{y})$ is the output error signal.

$\delta^{(2)}$
how wrong the output is; the negative residual
$(a^{(1)})^T$
what the hidden layer produced (transposed for the outer product)
Interpretation
the gradient is the outer product of the error and the hidden activation

Hidden Weights ($W_1$)

The chain from $J$ to $W_1$ is longer (must go through $W_2$):

Hidden Weight Gradient

$$\nabla_{W_1} J = \bigl[W_2^T \delta^{(2)} \odot \sigma'(z^{(1)})\bigr] \, x^T$$
$W_2^T \delta^{(2)}$
error propagated back through the output weights; blame distributed to each hidden neuron
$\sigma'(z^{(1)})$
how sensitive the activation is at the hidden layer; the derivative of the activation function evaluated at the pre-activation
$\odot$
element-wise multiplication (Hadamard product); each hidden neuron's error is scaled by its own activation sensitivity
$x^T$
the original input (transposed for the outer product)

Simplification for Linear Activation

For linear activation ($\sigma(z) = z$), the derivative is $\sigma'(z) = 1$, so the Hadamard product disappears and the gradient simplifies to:

$$\nabla_{W_1} J = W_2^T \delta^{(2)} x^T$$

This is the case used in the worked example below.

Part E: Weight Update (Gradient Descent)

Analogy: adjusting the factory workers.

After computing how much each worker contributed to the defect, you tell each one to adjust by a small amount proportional to their contribution. The learning rate $\eta$ controls how big the adjustment is (same as the step size $\alpha$ from Page 06).

Weight Update Rules

$$W_1 \leftarrow W_1 - \eta \nabla_{W_1} J$$ $$W_2 \leftarrow W_2 - \eta \nabla_{W_2} J$$
$\eta$
learning rate; controls the step size. Too large and you overshoot; too small and convergence is slow.
$-\nabla J$
the negative gradient direction; move opposite to the steepest ascent to decrease the loss

This Is Just Gradient Descent

This is exactly gradient descent from Page 06. The weights $W_1, W_2$ are the optimization variables (like $x$ in earlier chapters). The loss $J$ is the objective function. Backpropagation gives us $\nabla J$, and gradient descent moves in the direction $-\nabla J$. In practice, mini-batch SGD is used (compute gradients on a small batch of data, not the whole dataset), which introduces noise but converges faster for large datasets.

Part F: Worked Example (Full Numerical Computation)

You must show every matrix multiplication step. Follow the forward pass, compute the loss, backpropagate the error, and update all weights.

Problem Setup

  • Input: $x = [1, 0.5]^T$,   Target: $y = 1$
  • $W_1 = \begin{bmatrix}0.2 & 0.3\\0.1 & 0.4\end{bmatrix}$,   $W_2 = [0.5, 0.6]$
  • All biases = 0,   linear activation ($\sigma(z) = z$, $\sigma'(z) = 1$)
  • Learning rate: $\eta = 0.1$

Step 1: Forward Pass (Hidden Layer)

$$z^{(1)} = W_1 x = \begin{bmatrix}0.2 & 0.3\\0.1 & 0.4\end{bmatrix}\begin{bmatrix}1\\0.5\end{bmatrix} = \begin{bmatrix}0.2(1)+0.3(0.5)\\0.1(1)+0.4(0.5)\end{bmatrix} = \begin{bmatrix}0.35\\0.3\end{bmatrix}$$

Since $\sigma(z) = z$ (linear activation): $a^{(1)} = z^{(1)} = [0.35, 0.3]^T$

$a^{(1)} = [0.35, 0.3]^T$

Step 2: Forward Pass (Output)

$$\hat{y} = W_2 a^{(1)} = [0.5, 0.6]\begin{bmatrix}0.35\\0.3\end{bmatrix} = 0.5(0.35) + 0.6(0.3) = 0.175 + 0.18 = 0.355$$

$\hat{y} = 0.355$

Step 3: Loss

$$J = \frac{1}{2}(y - \hat{y})^2 = \frac{1}{2}(1 - 0.355)^2 = \frac{1}{2}(0.645)^2 = \frac{1}{2}(0.416025) \approx 0.208$$

$J \approx 0.208$

Step 4: Output Error

$$\delta^{(2)} = -(y - \hat{y}) = -(1 - 0.355) = -0.645$$

$\delta^{(2)} = -0.645$

Negative error signal means the output was too low (we need to increase $\hat{y}$ to reach $y = 1$).

Step 5: Output Gradient

$$\nabla_{W_2} J = \delta^{(2)} (a^{(1)})^T = (-0.645) \cdot [0.35, 0.3] = [-0.2258, -0.1935]$$

$\nabla_{W_2} J = [-0.2258, -0.1935]$

Step 6: Propagate Error to Hidden Layer

$$W_2^T \delta^{(2)} = \begin{bmatrix}0.5\\0.6\end{bmatrix}(-0.645) = \begin{bmatrix}-0.3225\\-0.387\end{bmatrix}$$

Backpropagated error = $[-0.3225, -0.387]^T$

Step 7: Hidden Gradient (linear activation, so $\sigma' = 1$)

Since $\sigma'(z) = 1$, the Hadamard product with the identity just gives us the backpropagated error directly:

$$\nabla_{W_1} J = \begin{bmatrix}-0.3225\\-0.387\end{bmatrix}[1, 0.5] = \begin{bmatrix}-0.3225 \times 1 & -0.3225 \times 0.5\\-0.387 \times 1 & -0.387 \times 0.5\end{bmatrix} = \begin{bmatrix}-0.3225 & -0.1613\\-0.387 & -0.1935\end{bmatrix}$$

$\nabla_{W_1} J = \begin{bmatrix}-0.3225 & -0.1613\\-0.387 & -0.1935\end{bmatrix}$

Step 8: Update Output Weights

$$W_2^{\text{new}} = W_2 - \eta \nabla_{W_2} J = [0.5, 0.6] - 0.1[-0.2258, -0.1935]$$ $$= [0.5 + 0.02258,\; 0.6 + 0.01935] = [0.5226, 0.6194]$$

$W_2^{\text{new}} = [0.5226, 0.6194]$

Step 9: Update Hidden Weights

$$W_1^{\text{new}} = W_1 - \eta \nabla_{W_1} J = \begin{bmatrix}0.2 & 0.3\\0.1 & 0.4\end{bmatrix} - 0.1\begin{bmatrix}-0.3225 & -0.1613\\-0.387 & -0.1935\end{bmatrix}$$ $$= \begin{bmatrix}0.2 + 0.03225 & 0.3 + 0.01613\\0.1 + 0.0387 & 0.4 + 0.01935\end{bmatrix} = \begin{bmatrix}0.2323 & 0.3161\\0.1387 & 0.4194\end{bmatrix}$$

$W_1^{\text{new}} = \begin{bmatrix}0.2323 & 0.3161\\0.1387 & 0.4194\end{bmatrix}$

Step 10: Verification

All weights increased. This makes sense: the network produced $\hat{y} = 0.355$ but needed $y = 1$, so it needed a larger output. Increasing the weights (which were all positive, multiplying positive inputs) pushes the output up toward the target. The gradients were all negative (pointing toward decreasing $J$), and subtracting a negative number increases the weight.

Glossary

Multi-Layer Perceptron (MLP)
A feedforward neural network with at least one hidden layer between the input and output. The simplest architecture for learning nonlinear relationships.
Hidden Layer
A layer of neurons between input and output. Its purpose is to learn intermediate representations; features the raw input does not explicitly contain.
Forward Pass
Computing the output $\hat{y}$ from the input $x$ by propagating values left to right through each layer.
Backpropagation
Computing $\nabla J$ with respect to every weight by applying the chain rule backwards through the network. The errors at the output are "propagated back" to assign blame to each weight.
Activation Function ($\sigma$)
A function applied after each layer's linear transformation. Common choices: sigmoid $\sigma(z) = 1/(1+e^{-z})$, ReLU $\sigma(z) = \max(0,z)$, or linear $\sigma(z) = z$. Without a nonlinear activation, the network collapses to a single linear model.
Loss Function ($J$)
A scalar measuring prediction error. MSE loss $J = \frac{1}{2}\|y - \hat{y}\|^2$ is standard for regression. Cross-entropy is standard for classification.
Learning Rate ($\eta$)
The step size for gradient descent. Analogous to $\alpha$ in Page 06. Controls how much each weight changes per update.
Epoch
One complete pass through the entire training dataset. Training typically runs for many epochs until the loss converges.
Mini-batch
A small random subset of the training data used to estimate the gradient. Introduces noise but makes each update much cheaper than computing the gradient over the full dataset.
