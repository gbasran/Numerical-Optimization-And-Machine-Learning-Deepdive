Part A: Gradient Descent & Newton

Gradient Descent Update $$x_{k+1} = x_k - \alpha_k \nabla f(x_k)$$ Move in the steepest downhill direction, step size $\alpha_k$.

Page 06

Exact Step Size on Quadratic $f(x) = \frac{1}{2}x^T A x - b^T x$ $$\alpha_k = \frac{g_k^T g_k}{g_k^T A g_k}$$ where $g_k = \nabla f(x_k) = Ax_k - b$. Minimizes $f$ exactly along the search direction.

Page 06

GD Convergence Rate on Quadratics $$f(x_{k+1}) - f(x^*) \leq \left(\frac{\kappa - 1}{\kappa + 1}\right)^2 \left(f(x_k) - f(x^*)\right)$$ where $\kappa = \lambda_{\max}/\lambda_{\min}$ is the condition number. Large $\kappa$ = slow convergence.

Page 06, Page 01

Newton Step $$p_k = -[\nabla^2 f(x_k)]^{-1} \nabla f(x_k)$$ Jump to the minimum of the local quadratic model. In practice, solve $\nabla^2 f(x_k)\, p_k = -\nabla f(x_k)$.

Page 07

Newton Update $$x_{k+1} = x_k + \alpha_k p_k$$ Use $\alpha_k = 1$ for pure Newton. With line search, $\alpha_k$ satisfies Wolfe/Armijo conditions.

Page 07

Quadratic Model $$m_k(p) = f(x_k) + \nabla f(x_k)^T p + \frac{1}{2} p^T B_k p$$ Second-order Taylor approximation of $f$ at $x_k$. $B_k = \nabla^2 f(x_k)$ for Newton.

Page 07, Page 09

Part B: Line Search Conditions

Armijo Condition (Sufficient Decrease) $$f(x_k + \alpha p_k) \leq f(x_k) + c_1 \alpha \nabla f(x_k)^T p_k$$ The function must decrease by at least a fraction $c_1$ of what the linear model predicts. Typical $c_1 = 10^{-4}$.

Page 08

Wolfe Curvature Condition $$\nabla f(x_k + \alpha p_k)^T p_k \geq c_2 \nabla f(x_k)^T p_k$$ The slope must flatten enough. Prevents stopping too early. Typical $c_2 = 0.9$ for Newton/quasi-Newton, $c_2 = 0.1$ for CG.

Page 08

Strong Wolfe Curvature Condition $$|\nabla f(x_k + \alpha p_k)^T p_k| \leq c_2 |\nabla f(x_k)^T p_k|$$ Absolute value version; prevents going too far past the minimum. Required for nonlinear CG.

Page 08

Backtracking Algorithm
  1. Start with $\alpha = \bar{\alpha}$ (e.g., $\bar{\alpha} = 1$)
  2. While Armijo condition is violated: $\alpha \leftarrow \rho \alpha$ (shrink by $\rho \in (0,1)$, typically $\rho = 0.5$)
  3. Return $\alpha$
Simple and effective. Only checks Armijo, not curvature. Use $\bar{\alpha} = 1$ for Newton/quasi-Newton.

Page 08

Part C: Trust Region

Trust-Region Subproblem $$\min_p \; m_k(p) = f_k + g_k^T p + \frac{1}{2} p^T B_k p \qquad \text{subject to } \|p\| \leq \Delta_k$$ Minimize the quadratic model inside a ball of radius $\Delta_k$.

Page 09

Trust-Region Ratio $$\rho_k = \frac{f(x_k) - f(x_k + p_k)}{m_k(0) - m_k(p_k)} = \frac{\text{actual reduction}}{\text{predicted reduction}}$$ Measures how well the model predicts reality. $\rho \approx 1$ = great model; $\rho \lt 0$ = model got it wrong.

Page 09

Radius Update Rules (typical)
  • If $\rho_k \lt 0.25$: shrink; $\Delta_{k+1} = 0.25\, \Delta_k$
  • If $\rho_k \gt 0.75$ and $\|p_k\| = \Delta_k$ (hit boundary): expand; $\Delta_{k+1} = \min(2\Delta_k, \Delta_{\max})$
  • Otherwise: keep; $\Delta_{k+1} = \Delta_k$
Accept step if $\rho_k \gt \eta$ (e.g., $\eta = 0.25$). Reject step if $\rho_k \leq \eta$ (keep $x_k$, shrink $\Delta$).

Page 09

Cauchy Point $$p^C = -\tau_k \frac{\Delta_k}{\|g_k\|} g_k$$ where $\tau_k = \begin{cases} 1 & \text{if } g_k^T B_k g_k \leq 0 \\ \min\left(\frac{\|g_k\|^3}{\Delta_k\, g_k^T B_k g_k},\; 1\right) & \text{otherwise} \end{cases}$ The steepest descent step capped to the trust region. Guarantees sufficient model decrease.

Page 09

Part D: Conjugate Gradient

Linear CG Algorithm (for $Ax = b$ with SPD $A$)
  1. Init: $r_0 = b - Ax_0$, $d_0 = r_0$
  2. Step size: $\alpha_k = \dfrac{r_k^T r_k}{d_k^T A d_k}$
  3. Update solution: $x_{k+1} = x_k + \alpha_k d_k$
  4. Update residual: $r_{k+1} = r_k - \alpha_k A d_k$
  5. Conjugacy param: $\beta_k = \dfrac{r_{k+1}^T r_{k+1}}{r_k^T r_k}$
  6. New direction: $d_{k+1} = r_{k+1} + \beta_k d_k$
Converges in at most $n$ steps. Residuals are orthogonal ($r_i^T r_j = 0$). Directions are $A$-conjugate ($d_i^T A d_j = 0$).

Page 10

Fletcher-Reeves (Nonlinear CG) $$\beta_k^{FR} = \frac{\|\nabla f(x_{k+1})\|^2}{\|\nabla f(x_k)\|^2} = \frac{g_{k+1}^T g_{k+1}}{g_k^T g_k}$$ Same as linear CG formula. Simple, but may generate poor directions on non-quadratic problems.

Page 11

Polak-Ribière (Nonlinear CG) $$\beta_k^{PR} = \frac{g_{k+1}^T (g_{k+1} - g_k)}{g_k^T g_k} = \frac{g_{k+1}^T g_{k+1} - g_{k+1}^T g_k}{\|g_k\|^2}$$ Uses gradient difference. Automatically restarts (gives $\beta \approx 0$) when the gradient doesn't change direction much. Generally preferred over FR.

Page 11

Nonlinear CG Direction Update $$d_{k+1} = -g_{k+1} + \beta_k d_k$$ Start with steepest descent ($-g$), then add a fraction of the previous direction. Use Strong Wolfe line search.

Page 11

Part E: Quasi-Newton Methods

Secant Condition $$B_{k+1} s_k = y_k \qquad \Longleftrightarrow \qquad H_{k+1} y_k = s_k$$ where $s_k = x_{k+1} - x_k$ (step) and $y_k = \nabla f(x_{k+1}) - \nabla f(x_k)$ (gradient change).
The update must match the observed gradient change. This is the defining property of quasi-Newton methods.

Page 12

Curvature Condition $$s_k^T y_k \gt 0$$ Required for BFGS to maintain positive definiteness. Guaranteed by Wolfe line search.

Page 12

BFGS Inverse Hessian Update $$H_{k+1} = (I - \rho_k s_k y_k^T)\, H_k\, (I - \rho_k y_k s_k^T) + \rho_k s_k s_k^T$$ where $\rho_k = \dfrac{1}{y_k^T s_k}$. Rank-2 update. Preserves PD if $s_k^T y_k \gt 0$.

Page 12

BFGS Direct Hessian Update $$B_{k+1} = B_k - \frac{B_k s_k s_k^T B_k}{s_k^T B_k s_k} + \frac{y_k y_k^T}{y_k^T s_k}$$ Updates $B_k \approx \nabla^2 f$. Rank-2 correction: subtract a rank-1 term, add a rank-1 term.

Page 12

SR1 Inverse Hessian Update $$H_{k+1} = H_k + \frac{(s_k - H_k y_k)(s_k - H_k y_k)^T}{(s_k - H_k y_k)^T y_k}$$ Rank-1 update. Can capture indefinite Hessians (unlike BFGS). Use with trust region.

Page 13

SR1 Direct Hessian Update $$B_{k+1} = B_k + \frac{(y_k - B_k s_k)(y_k - B_k s_k)^T}{(y_k - B_k s_k)^T s_k}$$ Rank-1 correction. Safeguard: skip update if $|(y_k - B_k s_k)^T s_k| \lt \epsilon \|s_k\| \|y_k - B_k s_k\|$.

Page 13

DFP Inverse Hessian Update $$H_{k+1} = H_k - \frac{H_k y_k y_k^T H_k}{y_k^T H_k y_k} + \frac{s_k s_k^T}{y_k^T s_k}$$ Dual of BFGS (swap $s \leftrightarrow y$, $H \leftrightarrow B$). BFGS generally preferred in practice.

Page 13

Quasi-Newton Search Direction $$d_k = -H_k \nabla f(x_k)$$ Same form as Newton ($d_k = -[\nabla^2 f]^{-1} g_k$) but using the approximation $H_k$ instead of the true inverse Hessian.

Page 12

Part F: Nonlinear Equations

Newton Step for Systems $r(x) = 0$ $$J(x_k)\, p_k = -r(x_k), \qquad x_{k+1} = x_k + p_k$$ $J(x_k)$ is the Jacobian matrix. Solve the linear system for $p_k$. Locally quadratic convergence.

Page 14

Jacobian $$J(x) = \begin{bmatrix} \dfrac{\partial r_1}{\partial x_1} & \cdots & \dfrac{\partial r_1}{\partial x_n} \\[6px] \vdots & \ddots & \vdots \\[6px] \dfrac{\partial r_m}{\partial x_1} & \cdots & \dfrac{\partial r_m}{\partial x_n} \end{bmatrix}$$ Row $i$, column $j$ is $\partial r_i / \partial x_j$. For optimization, $r = \nabla f$ and $J = \nabla^2 f$.

Page 02, Page 14

Broyden's Method (rank-1 Jacobian update) $$J_{k+1} = J_k + \frac{(y_k - J_k s_k)\, s_k^T}{s_k^T s_k}$$ where $s_k = x_{k+1} - x_k$ and $y_k = r(x_{k+1}) - r(x_k)$. Avoids recomputing Jacobian each step. Superlinear convergence.

Page 14

Scalar Newton (1D root finding) $$x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)}$$ Find $x$ where $f(x) = 0$ using tangent line intersection. Special case of the vector version.

Page 14

Part G: Constrained Optimization

Lagrangian (general) $$\mathcal{L}(x, \lambda, \mu) = f(x) - \sum_{i} \lambda_i c_i^{eq}(x) - \sum_{j} \mu_j c_j^{ineq}(x)$$ Combines objective and constraints into one function. $\lambda_i$ are equality multipliers, $\mu_j$ are inequality multipliers.

Page 15

KKT Conditions (for $\min f$ s.t. $c_i^{eq} = 0$, $c_j^{ineq} \geq 0$)
  1. Stationarity: $\nabla f(x^*) = \sum_i \lambda_i \nabla c_i^{eq}(x^*) + \sum_j \mu_j \nabla c_j^{ineq}(x^*)$
  2. Primal feasibility: $c_i^{eq}(x^*) = 0$, $\;c_j^{ineq}(x^*) \geq 0$
  3. Dual feasibility: $\mu_j \geq 0$ for all inequality constraints
  4. Complementary slackness: $\mu_j \cdot c_j^{ineq}(x^*) = 0$ for each $j$
Necessary conditions at any local min (under LICQ). For convex problems, also sufficient.

Page 15

Linear Programming Standard Form $$\min\; c^T x \qquad \text{s.t.}\; Ax = b, \; x \geq 0$$ Objective is linear, constraints are linear. Optimal solution always at a vertex of the feasible region.

Page 16

Barrier Function (Interior Point) $$\min_x \; c^T x - \mu \sum_{i=1}^{n} \ln(x_i) \qquad \text{s.t.}\; Ax = b$$ The $-\mu \ln(x_i)$ terms create a "repulsive force" near the boundaries $x_i = 0$. As $\mu \to 0$, the solution approaches the LP optimum.

Page 16

KKT for LP $$A^T \lambda + s = c, \qquad Ax = b, \qquad x \geq 0, \qquad s \geq 0, \qquad x_i s_i = 0$$ $s$ are slack variables for the dual. Complementary slackness: $x_i s_i = 0$ means at each variable, either $x_i = 0$ or $s_i = 0$ (or both).

Page 16

Part H: Key Definitions

Vector Norms $$\|x\|_1 = \sum_i |x_i|, \qquad \|x\|_2 = \sqrt{\sum_i x_i^2}, \qquad \|x\|_\infty = \max_i |x_i|$$ L1 = taxicab distance, L2 = straight-line distance, L-infinity = max component.

Page 01

Condition Number $$\kappa(A) = \frac{\lambda_{\max}(A)}{\lambda_{\min}(A)}$$ Ratio of largest to smallest eigenvalue (for SPD $A$). Large $\kappa$ = ill-conditioned = GD is slow = elliptical contours.

Page 01, Page 06

Convergence Rates
  • Q-linear: $\|e_{k+1}\| \leq r \|e_k\|$ for some $r \in (0,1)$. Error shrinks by constant factor. (GD)
  • Q-superlinear: $\|e_{k+1}\|/\|e_k\| \to 0$. Factor improves each step. (BFGS)
  • Q-quadratic: $\|e_{k+1}\| \leq C \|e_k\|^2$. Error squares each step. (Newton)
where $e_k = x_k - x^*$ is the error.

Page 01

Positive Definite Matrix
  • All eigenvalues $\gt 0$
  • Equivalently: $x^T A x \gt 0$ for all $x \neq 0$
  • Equivalently: all leading principal minors $\gt 0$
PD Hessian = strict local minimum. PD matrix = CG works. PD $H_k$ = descent direction guaranteed.

Page 03

Convexity $$f(\alpha x + (1-\alpha)y) \leq \alpha f(x) + (1-\alpha) f(y) \quad \forall\, \alpha \in [0,1]$$ The function lies below the chord between any two points. Equivalently: $\nabla^2 f(x) \succeq 0$ everywhere (PSD Hessian). For convex functions, every local minimum is a global minimum.

Page 05

Taylor Expansion (multivariable, to 2nd order) $$f(x + p) \approx f(x) + \nabla f(x)^T p + \frac{1}{2} p^T \nabla^2 f(x)\, p$$ The foundation of Newton's method and the quadratic model. First term = value, second = slope, third = curvature.

Page 01, Page 07

$2 \times 2$ Matrix Inverse $$\begin{bmatrix}a & b\\c & d\end{bmatrix}^{-1} = \frac{1}{ad - bc}\begin{bmatrix}d & -b\\-c & a\end{bmatrix}$$ Swap diagonals, negate off-diagonals, divide by determinant. Essential for every exam computation.

Page 03

$2 \times 2$ Eigenvalues $$\lambda = \frac{(a+d) \pm \sqrt{(a-d)^2 + 4bc}}{2} \qquad \text{for } A = \begin{bmatrix}a&b\\c&d\end{bmatrix}$$ From the characteristic equation $\det(A - \lambda I) = 0$. For symmetric matrices, eigenvalues are always real.
