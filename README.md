## Design

Every page follows the same structure:

- Physical analogy first, then math
- Every formula has every symbol explained
- Hand computations mirror actual quiz and assignment problems
- Cross-references link back whenever prior concepts are used
- Glossary at the bottom of each page

Textbook: Nocedal & Wright, *Numerical Optimization*, 2nd Edition
