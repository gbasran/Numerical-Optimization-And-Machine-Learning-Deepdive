Convexity: What Makes Problems Nice

Convexity is the single most important structural property in optimization; it guarantees local = global

Part A: Convex Sets

Physical analogy; A fenced yard with no holes or indentations.

Picture a fenced yard. Pick any two points inside the yard. Now walk in a perfectly straight line from one point to the other. If you NEVER leave the yard; no matter which two points you pick; the yard is a convex set.

Convex examples: A circular yard. A rectangular yard. A triangular yard. Any regular shape without dents.

Non-convex examples: An L-shaped yard. A star-shaped yard. A crescent shape. In each of these, you can find two interior points where the straight line between them passes OUTSIDE the yard.

The key mental image: convex = "no indentations." If you push the boundary inward anywhere, you break convexity.
Definition: Convex Set $$\text{A set } C \text{ is convex if } \alpha x + (1 - \alpha)y \in C \text{ for all } x, y \in C \text{ and all } \alpha \in [0, 1]$$
$C$
the set (the "yard")
$x, y$
any two points inside $C$ (any two spots in the yard)
$\alpha$
a number between 0 and 1; it parameterizes the straight line from $y$ (when $\alpha = 0$) to $x$ (when $\alpha = 1$)
$\alpha x + (1 - \alpha)y$
a point on the straight line segment between $x$ and $y$; this is called a convex combination
$\in C$
the point must remain inside the set

When $\alpha = 0$: you get $y$. When $\alpha = 1$: you get $x$. When $\alpha = 0.5$: you get the midpoint. As $\alpha$ varies from 0 to 1, you trace the entire line segment from $y$ to $x$.

Why do we care about convex sets?
In constrained optimization (Page 15), the set of feasible points is often a convex set. If the feasible region is convex AND the objective function is convex, then the problem is a "convex optimization problem"; and finding the optimum is dramatically easier.

Part B: Convex Functions; Properties and Tests

Physical analogy; The rubber band test.

Take a convex function and plot its graph. Now pick any two points on the curve and stretch a rubber band between them. The rubber band forms a straight line above those two points.

If the rubber band ALWAYS stays above (or touches) the curve between the two points; no matter which two points you choose; the function is convex.

If the curve ever pokes ABOVE the rubber band, even at a single spot, the function is not convex.

Visually: a convex function "sags" between any two points. A non-convex function has bumps or valleys that poke above the rubber band.
Test 1: The Definition (Rubber Band Test) $$f(\alpha x + (1-\alpha)y) \leq \alpha f(x) + (1-\alpha) f(y) \quad \text{for all } x, y \text{ and } \alpha \in [0,1]$$
Left side
$f$ evaluated at a point on the line segment between $x$ and $y$; the function value on the curve
Right side
the corresponding point on the straight line (rubber band) between $f(x)$ and $f(y)$
$\leq$
the curve is below (or touching) the rubber band

This is the most fundamental definition, but it is hard to check in practice because you would need to verify it for ALL pairs $x, y$. That is why we have equivalent tests that are easier to use.

Test 2: First-Order Characterization (Tangent Line Test)

If $f$ is differentiable, then $f$ is convex if and only if:

$$f(y) \geq f(x) + \nabla f(x)^T (y - x) \quad \text{for all } x, y$$
$f(x) + \nabla f(x)^T(y-x)$
the first-order Taylor approximation of $f$ at $x$; the tangent line (or tangent plane in multiple dimensions)
$f(y) \geq \ldots$
the actual function value at $y$ is ALWAYS at or above the tangent approximation

In words: "A convex function always lies above all of its tangent lines." If you draw the tangent line at any point, the function never dips below it. This is the "supported from below" property.

Recall from Page 01 that the first-order Taylor expansion gives you the tangent approximation. For convex functions, this approximation is always an underestimate.

Test 3: Second-Order Characterization (Hessian Test)

If $f$ is twice differentiable, then $f$ is convex if and only if:

$$\nabla^2 f(x) \text{ is positive semidefinite (PSD) for ALL } x$$
$\nabla^2 f(x)$
the Hessian matrix (from Page 02) evaluated at point $x$
PSD for ALL $x$
no matter WHERE you evaluate the Hessian, all eigenvalues are $\geq 0$ (from Page 03)

In words: "The curvature is non-negative in every direction, at every point." The function never curves downward anywhere. This is the easiest test to apply in practice.

Strictly convex: If $\nabla^2 f(x)$ is PD (not just PSD) for all $x$, then $f$ is strictly convex. This means the bowl has no flat directions; it curves strictly upward everywhere.

Summary of the three tests; all equivalent:

TestWhat It ChecksWhen to Use
Definition (rubber band)$f(\alpha x + (1-\alpha)y) \leq \alpha f(x) + (1-\alpha)f(y)$Theoretical proofs
First-order (tangent)$f(y) \geq f(x) + \nabla f(x)^T(y-x)$When you only have gradients
Second-order (Hessian)$\nabla^2 f(x) \succeq 0$ for all $x$Most practical; check Hessian PSD

The $\succeq 0$ notation means "positive semidefinite" (all eigenvalues $\geq 0$).

Part C: Why Convexity Is THE Big Deal

Physical analogy; A landscape with exactly one valley.

Imagine a landscape that is one giant bowl. No hidden valleys, no plateaus, no secondary dips. Just one big bowl going down to a single lowest point.

If you are a hiker trying to find the lowest elevation, life is easy: just keep walking downhill. No matter where you start, no matter which downhill path you take, you will eventually reach THE bottom. There are no tricks, no trap doors, no false bottoms.

Now imagine a real mountain range with thousands of valleys, ridges, and plateaus. You walk downhill and find a low point. But is it THE lowest point, or just A low point? You have no way to know without exploring the entire landscape.

Convex functions are the first landscape. Non-convex functions are the second. This is why convexity is such a powerful property; it turns an impossibly hard search problem into a straightforward one.
The Key Theorem: Local = Global for Convex Functions $$\text{If } f \text{ is convex and } \nabla f(x^*) = \mathbf{0}, \text{ then } x^* \text{ is a GLOBAL minimizer.}$$

What this means:

  • You do not need to search further. There are no other, lower points anywhere.
  • Any local minimum is automatically the global minimum.
  • If $f$ is strictly convex, the global minimizer is UNIQUE; there is exactly one lowest point.

Why this is powerful for algorithms: If gradient descent (Page 06) finds a stationary point of a convex function, it has found THE answer, not just A answer. No need to restart from different initial points.

Connection to Quadratics

The quadratic function $f(\mathbf{x}) = \frac{1}{2}\mathbf{x}^T Q \mathbf{x} + \mathbf{b}^T \mathbf{x} + c$ is convex if and only if $Q$ is positive semidefinite (PSD).

$Q$
a symmetric matrix that determines the curvature (the "shape of the bowl")
$\mathbf{b}$
a vector that shifts the bowl's center
$c$
a scalar that shifts the function value up or down

Why? The Hessian of this function is $\nabla^2 f = Q$ (constant , does not depend on $\mathbf{x}$). So convexity requires $Q$ to be PSD. If $Q$ is PD (strictly positive definite), the function is strictly convex; a perfect bowl with a unique minimum.

This is exactly why every quiz problem uses quadratics with PD matrices: they give us clean, well-behaved test cases.

Hand Computation: Prove $f(\mathbf{x}) = \frac{1}{2}\mathbf{x}^T Q \mathbf{x}$ Is Convex

Let $Q = \begin{bmatrix} 2 & 0 \\ 0 & 3 \end{bmatrix}$ and $f(\mathbf{x}) = \frac{1}{2}\mathbf{x}^T Q \mathbf{x}$.

Step 1: Compute the Hessian
For a quadratic $f(\mathbf{x}) = \frac{1}{2}\mathbf{x}^T Q \mathbf{x}$, the gradient is: $$\nabla f(\mathbf{x}) = Q\mathbf{x}$$ and the Hessian is: $$\nabla^2 f(\mathbf{x}) = Q = \begin{bmatrix} 2 & 0 \\ 0 & 3 \end{bmatrix}$$ The Hessian is constant (it does not depend on $\mathbf{x}$).
Step 2: Check PSD
We need all eigenvalues of $Q$ to be $\geq 0$.

Since $Q$ is diagonal, the eigenvalues are the diagonal entries (recall from Page 03, Hand Computation 1): $$\lambda_1 = 2 > 0, \quad \lambda_2 = 3 > 0$$
Step 3: Conclude
Both eigenvalues are strictly positive, so $Q$ is positive definite (which is stronger than PSD).
Result: $\nabla^2 f = Q$ is PD for all $\mathbf{x}$, so $f$ is strictly convex. Any stationary point is the unique global minimizer.
What is that minimizer? Set $\nabla f = Q\mathbf{x} = \mathbf{0}$. Since $Q$ is invertible ($\det = 6 \neq 0$), the only solution is $\mathbf{x}^* = \mathbf{0}$. And $f(\mathbf{0}) = 0$. So the unique global minimum is at the origin with value 0.

Why Every Quiz Uses Convex Quadratics

Every test problem in Quizzes 4, 5, and 6 uses convex quadratics. This is not a coincidence. Here is why:

  • Clean test cases: Convex quadratics have a unique minimum we can compute exactly, so we can verify whether an algorithm gives the right answer.
  • Exact arithmetic: On quadratics, some algorithms converge in a finite number of steps (e.g., conjugate gradient converges in exactly $n$ steps → see Page 10).
  • No globalization needed: Since local = global, we do not need line search or trust regions to guarantee convergence; the quadratic model IS the true function.
  • Foundation for theory: Newton's method (Page 07) works by approximating any function with a quadratic. Understanding quadratics = understanding Newton.

Part D: Non-Convex Reality

The real world is not convex.

The nice, clean, single-bowl quadratics on quizzes are training wheels. Real optimization problems are almost never convex. They have multiple valleys, ridges, saddle points, and plateaus. But the theory and algorithms we learn on convex problems carry over; they just need extra safeguards.
Example: The Rosenbrock Function (Assignment 1) $$f(x_1, x_2) = 100(x_2 - x_1^2)^2 + (1 - x_1)^2$$

This is a famous test function. Its minimum is at $(1, 1)$ with $f = 0$, but it is notoriously hard to minimize because:

  • It has a narrow, curved valley (the "banana valley")
  • The valley floor is nearly flat, so algorithms stall
  • Gradient descent takes thousands of steps to converge

Is it convex? Let us check. The Hessian is:

$$\nabla^2 f = \begin{bmatrix} 1200x_1^2 - 400x_2 + 2 & -400x_1 \\ -400x_1 & 200 \end{bmatrix}$$

At $(0, 0)$: $H = \begin{bmatrix} 2 & 0 \\ 0 & 200 \end{bmatrix}$. Eigenvalues: 2 and 200. PD here.

At $(0, 1)$: $H = \begin{bmatrix} -398 & 0 \\ 0 & 200 \end{bmatrix}$. Eigenvalues: $-398$ and 200. Indefinite!

Since the Hessian is NOT PSD everywhere, the Rosenbrock function is not convex.

Neural networks are extremely non-convex.
If you have heard of deep learning, here is a connection: the loss function of a neural network has millions of parameters and is wildly non-convex. It has countless local minima, saddle points, and plateaus. Yet optimization algorithms (essentially fancy versions of gradient descent) still find good solutions. This is an active area of research; why does gradient descent work so well on non-convex problems?
Why Algorithms Still Work on Non-Convex Problems

When a problem is non-convex, algorithms cannot guarantee finding the GLOBAL minimum. But they can reliably find LOCAL minima, which are often good enough. The key strategies are:

StrategyWhat It DoesPage
Line searchEnsures each step makes sufficient progress (does not overshoot)Page 08
Trust regionLimits step size to a "safe zone" where the quadratic model is reliablePage 09
Multiple startsRun the algorithm from many different starting points and keep the bestGeneral technique
PD enforcementBFGS maintains a PD Hessian approximation, avoiding saddle directionsPage 12

These are called globalization strategies; they help algorithms navigate non-convex landscapes safely. Line search and trust regions are not about finding the global minimum; they are about ensuring the algorithm does not blow up or get stuck.

The Convexity Hierarchy; From Nicest to Hardest

Problem TypePropertyDifficulty
Strongly convex quadraticPD Hessian, constant curvatureEasiest; CG solves exactly in $n$ steps
Strongly convex (general)PD Hessian everywhere, but not quadraticEasy; local = global, fast convergence
Convex (not strongly)PSD Hessian everywhereMedium; local = global but may have flat directions

The course quizzes work at the top (strongly convex quadratics). The assignments (Rosenbrock) work at the bottom (non-convex). The theory in between explains why the algorithms bridge these worlds.

Part E: Convexity Preserving Operations

Building Convex Functions from Simpler Ones

These rules let you prove a function is convex by breaking it into simpler pieces:

OperationRuleExample
Non-negative sum$\alpha f + \beta g$ is convex if $f, g$ convex and $\alpha, \beta \geq 0$$3x^2 + 5e^x$ (both terms convex)
Pointwise maximum$\max(f, g)$ is convex if $f, g$ convex$\max(x, -x) = |x|$
Composition with affine$f(Ax + b)$ is convex if $f$ convex$\|Ax - b\|^2$ (least squares)
Restriction to a line$g(t) = f(x + t\mathbf{d})$ is convex in $t$ if $f$ is convexLine search on a convex function

Why this matters: In Page 08 (Line Search), we minimize along a line: $\phi(\alpha) = f(x_k + \alpha d_k)$. If $f$ is convex, then $\phi$ is convex in $\alpha$, so the line search has a unique minimum; the Wolfe conditions are easier to satisfy.

Hand Computation: Is $f(x_1, x_2) = x_1^2 + 4x_2^2 + 2x_1 x_2$ Convex?

Step 1: Write as a quadratic
$$f(\mathbf{x}) = \frac{1}{2}\mathbf{x}^T Q \mathbf{x} \quad \text{where } Q = \begin{bmatrix} 2 & 2 \\ 2 & 8 \end{bmatrix}$$ (Note: we need $Q$ symmetric, so the off-diagonal is $2 \cdot 1 = 2$ from the cross term $2x_1 x_2$, and diagonal entries doubled from the $x_1^2$ and $4x_2^2$ terms.)
Step 2: Check eigenvalues of $Q$
Trace $= 2 + 8 = 10$, $\det = 16 - 4 = 12$. $$\lambda^2 - 10\lambda + 12 = 0$$ $$\lambda = \frac{10 \pm \sqrt{100 - 48}}{2} = \frac{10 \pm \sqrt{52}}{2} = \frac{10 \pm 2\sqrt{13}}{2} = 5 \pm \sqrt{13}$$ $\sqrt{13} \approx 3.61$, so $\lambda_1 \approx 8.61 > 0$ and $\lambda_2 \approx 1.39 > 0$.
Result: Both eigenvalues are positive, so $Q$ is PD, so $f$ is strictly convex. Any local minimum is the unique global minimum.
Quick check with 2×2 shortcut: $a_{11} = 2 > 0$ and $\det = 12 > 0$. PD confirmed without computing eigenvalues.
Common convex functions you should recognize on sight:
• $f(x) = x^2$; convex (parabola opens upward)
• $f(x) = |x|$; convex (V-shape)
• $f(x) = e^x$; convex (exponential curves upward)
• $f(x) = -\ln(x)$ for $x > 0$; convex (negative log curves upward)
• $f(\mathbf{x}) = \|\mathbf{x}\|^2$; convex (sum of squares is always a bowl)
• $f(\mathbf{x}) = \frac{1}{2}\mathbf{x}^T Q \mathbf{x}$ with $Q$ PSD; convex

Common non-convex functions:
• $f(x) = \sin(x)$; not convex (oscillates)
• $f(x) = x^3$; not convex (inflection point at 0)
• Rosenbrock function; not convex (see Part D above)
• Any function with multiple local minima; not convex

Exam Strategy for Convexity Questions

When an exam question asks "Is this function convex?", here is your game plan:

  1. If it is a quadratic $f(\mathbf{x}) = \frac{1}{2}\mathbf{x}^T Q \mathbf{x} + \mathbf{b}^T \mathbf{x} + c$: check if $Q$ is PSD. If yes, convex. If $Q$ is PD, strictly convex.
  2. If it is a general function: compute the Hessian $\nabla^2 f$. Check if it is PSD for ALL $\mathbf{x}$ (not just at one point).
  3. Shortcut for 2×2: the Hessian is PSD everywhere if $a_{11}(\mathbf{x}) \geq 0$ and $\det(\nabla^2 f(\mathbf{x})) \geq 0$ for all $\mathbf{x}$.
  4. If you can decompose it: use the preservation rules above (sum of convex functions is convex, etc.).

Glossary

Convex Set
A set where the line segment between any two points stays inside the set. No indentations. $\alpha x + (1-\alpha)y \in C$ for all $x, y \in C$ and $\alpha \in [0,1]$.
Convex Function
A function where the rubber band always stays above (or touches) the curve: $f(\alpha x + (1-\alpha)y) \leq \alpha f(x) + (1-\alpha)f(y)$. Equivalently: Hessian is PSD everywhere.
Strictly Convex
A convex function with strict inequality (the rubber band is strictly above the curve, never touching except at the endpoints). Hessian is PD everywhere. Has at most one minimizer.
Strongly Convex
A function whose Hessian eigenvalues are bounded away from zero: $\nabla^2 f(x) \succeq mI$ for some $m > 0$. The "bowl" has a minimum curvature. Guarantees fast convergence rates.
Convex Combination
$\alpha x + (1-\alpha)y$ for $\alpha \in [0,1]$. A weighted average of two points that always lies on the line segment between them.
Global Minimizer
The point with the smallest function value over the entire domain. For convex functions, any local minimizer is automatically global.
First-Order Characterization
$f$ is convex iff $f(y) \geq f(x) + \nabla f(x)^T(y-x)$. The function always lies above its tangent lines.
Second-Order Characterization
$f$ is convex iff $\nabla^2 f(x) \succeq 0$ (PSD) for all $x$. No negative curvature anywhere.
Non-Convex
A function that is not convex. May have multiple local minima, saddle points, and regions of negative curvature. Most real-world problems are non-convex.
